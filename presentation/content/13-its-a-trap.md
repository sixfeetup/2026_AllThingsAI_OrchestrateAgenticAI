DEMO TIME — contract analysis and adversarial review.

This demo shows the full stack of agentic orchestration:

1. Ingest the consulting contract PDF — agent generates visual
   explainers and a review checklist automatically
2. Security as a first-class requirement: show the security skills
   (risk review, guardrails) that are baked into the workflow
3. Swarm with /c^2 — fan out to Codex and Gemini for adversarial
   review. Multiple models catch what a single model misses.
4. Show the visual explainers: project overlay, risk report, and
   adversarial findings side by side
5. Fire up speckit: turn findings into actionable specs, clarify
   ambiguities, generate a plan

Key points: this is multi-agent, multi-model orchestration. Security
isn't an afterthought — it's a skill the agent uses automatically.
The Gygax quote lands here: the trap isn't the contract, it's
skipping the review.
