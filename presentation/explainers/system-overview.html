4 Skills Ecosystem
build /render-deck
Build and open the deck. Runs uv run build.py --open.
watch /deck-watch
Watcher on plan.md + content. Processes @@ directives, regenerates slides, rebuilds. Use with /loop 90s.
image /gen-image
Generate images via Gemini or DALL-E. Flags for style, size, backend, output path.
parallel /c^2
Dispatch parallel workers in isolated worktrees. Swarm for fan-out, gather for collection. PRs per worker.
secrets /aix
Cache API keys from 1Password via op run. One biometric auth per session.
visual /visual-explainer
Generate self-contained HTML pages for diagrams, architecture, data tables. This page is one.
notify /say
macOS say for audio alerts. Used at milestones, blockers, and when ready for input. Fun voices.
bg /bg
Run and manage background shell jobs. Alias for /job. Launch, list, stop, check status.
loop /loop
Schedule recurring prompts via CronCreate. /loop 90s /deck-watch for the watcher cycle.
5 Deployment
GitHub Actions (automatic)
  • Triggers on push to main (presentation/, images/, plan.md)
  • Installs uv via astral-sh/setup-uv
  • Runs uv run build.py
  • Stages _site/ with deck + referenced images only
  • Deploys via actions/deploy-pages
  • Also available via workflow_dispatch (manual trigger)
Local (make deploy)
  • dist/ is a git worktree on gh-pages branch
  • make build writes to both output/ and dist/
  • make deploy copies referenced images, commits, pushes
  • make serve runs python3 -m http.server from dist/
  • Image grep: only images referenced in deck.html are copied
6 Key Conventions
Image Paths

Quotes — Blockquotes are extracted from slide content flow and positioned absolutely at the bottom-right of each slide, just left of the page number. CSS: .slide-quote wraps the blockquote with position: absolute; bottom; right.
Speaker Notes — Add ??? on its own line. Everything below becomes an <aside class="speaker-notes">. Press N during presentation to toggle the notes overlay (bottom-right, dark panel).
Explainer Links — Use [explainer: Title](path.html) in slide markdown. Renders as a styled teal pill button that opens in a new tab. Store explainer HTML in presentation/explainers/.
Image Optimization — Large PNGs get resized to *_web.jpg (max 1400px, JPEG quality 82) via Pillow. Only web-optimized images should be referenced in slides. Deploy only copies referenced images (~2MB total).
